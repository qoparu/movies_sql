# Анализ фильмов и сериалов (IMDb/Kaggle)

## Описание проекта
Проект анализирует данные о фильмах, актерах, жанрах и рейтингах. Цели:
- Топ-10 фильмов по рейтингу.
- Самые популярные жанры.
- Связь между бюджетом и рейтингом.
